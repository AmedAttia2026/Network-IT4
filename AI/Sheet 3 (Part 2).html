Part A - True / False Questions

Instructions: Write "T" for True or "F" for False.

1. The `match` statement in Python works similarly to the `switch` statement in other languages.
(T) - **Rationale:** It allows pattern matching against an expression.
2. In a `match` statement, each case can contain an `if` guard condition.
(T) - **Rationale:** The `if` guard adds an extra condition that must be met for the case to execute.
3. The membership operator `in` returns `True` if the value exists in a sequence.
(T)
4. The `is` operator compares two objects by value, not by memory location.
(F) - **Rationale:** The `is` operator compares object **identity** (memory location); `==` compares value.
5. The expression `'a' in ['a', 'b', 'c']` returns `True`.
(T)
6. The `not in` operator returns `True` if the item is not found in the given sequence.
(T)
7. In list comprehension, the `for` keyword must appear before the `if` condition.
(T) - **Rationale:** The structure is `[expression for item in iterable if condition]`.
8. The `match` statement can only compare numbers, not strings.
(F) - **Rationale:** `match` can compare various types, including numbers, strings, and complex patterns.
9. The `is` operator checks if two variables point to the same object in memory.
(T)
10. `[x**2 for x in range(4)]` produces `[1, 4, 9, 16]`.
(F) - **Rationale:** `range(4)` produces `0, 1, 2, 3`. Squaring these gives `[0, 1, 4, 9]`.

Part B - Multiple-Choice Questions

Instructions: Choose the correct answer for each question.

1. What will be the output of the following code?
x = 3
match x:
case 1: print("One")
case 2: print("Two")
case 3: print("Three")
a) One
b) Two
c) Three
d) Error
c) Three - **Rationale:** The value `x=3` matches the third case statement.
2. Which operator checks for **object identity**?
a) `==`
b) `is`
c) `in`
d) `!=`
b) `is`
3. What is the output of this code?
name = "Roa"
print("a" in name)
a) 'a'
b) True
c) False
d) Error
c) False - **Rationale:** The check is case-sensitive, and the string contains 'R', 'o', 'a', but not lowercase 'a'. (If the question intended 'a' to match the letter 'a' in 'Roa', the answer would be True, but strictly speaking, it is False). *Assuming strict case sensitivity.*
4. Which of the following is **not** a membership operator?
a) `in`
b) `not in`
c) `is`
d) None of the above
c) `is` - **Rationale:** `is` is an **identity** operator. `in` and `not in` are membership operators.
5. What is the output of the following list comprehension?
result = [x for x in range(5) if x % 2 == 0]
a) `[1, 3, 5]`
b) `[0, 2, 4]`
c) `[2, 4]`
d) `[0, 1, 2, 3, 4]`
b) `[0, 2, 4]` - **Rationale:** `range(5)` is `0, 1, 2, 3, 4`. Even numbers (modulo 2 equals 0) are 0, 2, and 4.

6. Which statement is true about the `is` operator?
a) It compares object identity
b) It compares values
c) It works only on numbers
d) It is the same as `==`
a) It compares object identity
7. What will be printed?
a = [1, 2, 3]
b = a
print(a is b)
a) True
b) False
c) Error
d) None
a) True - **Rationale:** `b = a` means both variables point to the *exact same* list object in memory.
8. Choose the correct list comprehension that generates squares of numbers from 1 to 5.
a) `[x^2 for x in range(1,6)]`
b) `[x**2 for x in range(1,6)]`
c) `[x*2 for x in range(1,6)]`
d) `(x**2 for x in range(1,6))`
b) `[x**2 for x in range(1,6)]` - **Rationale:** `**` is the exponentiation operator (square), and `range(1,6)` produces numbers 1 through 5.
9. What does this code print?
x = 10
y = 10
print(x is y)
a) True
b) False
c) Depends on Python version
d) Error
a) True - **Rationale:** Python caches small integers (typically -5 to 256) for optimization, meaning `x` and `y` reference the same object.
10. Which of the following statements is correct?
a) `match` replaces `if-else` completely.
b) The `match` statement must always have a default case.
c) The `case` statements are checked in order, top to bottom.
d) None of the above.
c) The `case` statements are checked in order, top to bottom.

Part C - Programming Tasks (Model Solutions)

Instructions: Write and test the following small programs.

1. Write a `match-case` program that prints “Excellent”, “Good”, or “Poor” based on a score value (A, B, or C).
score = input("Enter score (A, B, or C): ")
match score.upper():
case "A":
print("Excellent")
case "B":
print("Good")
case "C":
print("Poor")
case _:
print("Invalid score")
2. Write a Python program that checks if "apple" is in a list of fruits using a membership operator.
fruits = ["banana", "orange", "apple", "grape"]
if "apple" in fruits:
print("Apple found in the list.")
else:
print("Apple not found.")
3. Write a program to check if two variables `a` and `b` refer to the same object using the identity operator.
a = [10, 20]
b = a # b points to the same object as a
c = [10, 20] # c points to a new object, though value is the same

print(f"a is b: {a is b}") # True
print(f"a is c: {a is c}") # False
4. Use list comprehension to create a list of squares for all numbers from 1 to 10.
squares = [x**2 for x in range(1, 11)]
print(squares)
# Output: [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
5. Write a list comprehension that extracts only even numbers from `[1,2,3,4,5,6,7,8]`.
data = [1, 2, 3, 4, 5, 6, 7, 8]
even_numbers = [x for x in data if x % 2 == 0]
print(even_numbers)
# Output: [2, 4, 6, 8]
