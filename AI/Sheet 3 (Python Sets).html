Part A - True / False Questions

Instructions: Write T if the statement is true and F if the statement is false.

1. Sets in Python can contain duplicate elements.
(F) - **Rationale:** Sets are defined as collections of **unique** elements.
2. The `len()` function is used to count the number of items in a set.
(T)
3. Sets can hold items of different data types.
(T) - **Rationale:** Sets can hold heterogeneous data types (e.g., integers, strings, tuples).
4. Elements in a set can be accessed using an index.
(F) - **Rationale:** Sets are **unordered** and do not support indexing or slicing.
5. The `add()` method inserts a new element into a set.
(T)
6. The `remove()` method does not raise an error if the element doesn’t exist.
(F) - **Rationale:** The `remove()` method **does** raise a `KeyError`; the `discard()` method does not.
7. The `|` operator performs a union between two sets.
(T)
8. The intersection of two sets gives elements present in both.
(T)
9. A `frozenset` is a mutable type of set.
(F) - **Rationale:** A `frozenset` is an **immutable** version of a set.
10. Lists, tuples, or strings can be converted into sets using `set()`.
(T)

Part B - Multiple-Choice Questions

Instructions: Choose the correct answer for each question.

1. Which statement correctly creates an empty set?
a) `{}`
b) `set()`
c) `[]`
d) `set([])`
b) `set()` - **Rationale:** `{}` creates an empty dictionary, not an empty set.
2. What happens when a duplicate element is added to a set?
a) Raises an error
b) Ignored
c) Replaced
d) Duplicated
b) Ignored - **Rationale:** Sets automatically maintain uniqueness, so duplicates are silently discarded.
3. Which method removes and returns a random element?
a) `pop()`
b) `remove()`
c) `delete()`
d) `discard()`
a) `pop()` - **Rationale:** Since sets are unordered, `pop()` removes an arbitrary (random) element.
4. What does `A & B` represent?
a) Union
b) Difference
c) Intersection
d) Symmetric Difference
c) Intersection - **Rationale:** The `&` operator finds elements common to both sets.
5. Which method gives elements that exist in the first set but not in the second?
a) `difference()`
b) `union()`
c) `intersection()`
d) `symmetric_difference()`
a) `difference()` (or `A - B` operator).
6. Which operation gives items that are in either set, but not both?
a) `A | B`
b) `A & B`
c) `A - B`
d) `A ^ B`
d) `A ^ B` - **Rationale:** The caret operator (`^`) performs the symmetric difference operation.
7. The `discard()` method differs from `remove()` because it:
a) Removes duplicates only
b) Ignores missing elements
c) Raises an error
d) Adds elements
b) Ignores missing elements - **Rationale:** If the element is not found, `discard()` does nothing, while `remove()` raises a `KeyError`.
8. What is the output of `max({2, 4, 1, 5})`?
a) 2
b) 1
c) 4
d) 5
d) 5
9. Which of the following about `frozenset` is correct?
a) It is immutable
b) It allows duplicates
c) It can be changed
d) It uses `add()`
a) It is immutable - **Rationale:** `frozenset` is Python's immutable set type.
10. Converting `[1, 2, 2, 3]` into a set results in:
a) `{1, 2, 3}`
b) `{1, 2, 2, 3}`
c) `[1, 2, 3]`
d) `(1, 2, 3)`
a) `{1, 2, 3}` - **Rationale:** The list is converted, and the duplicate `2` is removed.

Part C - Programming Tasks (Model Solutions)

Instructions: Write and test the following small programs.

1. Create a set called `colors` with "red", "green", "blue" and add "yellow".
colors = {"red", "green", "blue"}
colors.add("yellow")
print(colors)
# Output: {'red', 'green', 'blue', 'yellow'} (order may vary)
2. Write code to find the intersection of `{1, 2, 3}` and `{3, 4, 5}`.
set_a = {1, 2, 3}
set_b = {3, 4, 5}
intersection_result = set_a & set_b # Using the operator
# OR: intersection_result = set_a.intersection(set_b)
print(intersection_result)
# Output: {3}
3. Safely remove "apple" from a set called `fruits` without raising an error.
fruits = {"banana", "orange", "grape"}
# Use discard() for safe removal
fruits.discard("apple") # Does nothing if 'apple' is missing
fruits.discard("banana") # Removes 'banana'
print(fruits)
# Output: {'orange', 'grape'} (order may vary)
4. Display the maximum and minimum values from `{15, 5, 25, 10}`.
data = {15, 5, 25, 10}
max_val = max(data)
min_val = min(data)
print(f"Maximum value: {max_val}")
print(f"Minimum value: {min_val}")
# Output: Maximum value: 25, Minimum value: 5
5. Convert the tuple `(2, 4, 4, 6, 8)` into a set and print it.
my_tuple = (2, 4, 4, 6, 8)
my_set = set(my_tuple)
print(my_set)
# Output: {8, 2, 4, 6} (order may vary, duplicates removed)
