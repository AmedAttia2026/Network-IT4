Model Answer: Data Types Operations Comparison

Comparison of operations on four fundamental Python data types: List, Tuple, Set, and Dictionary.

Operation (العملية) List (القائمة) Tuple (الثلاثية) Set (المجموعة) Dictionary (القاموس)
1. Accessing Items Yes (نعم) - By Index: `my_list[0]` Yes (نعم) - By Index: `my_tuple[0]` No (لا) - Unordered; requires iteration/conversion. Yes (نعم) - By Key: `my_dict['key']`
2. Searching for Duplicates Yes (نعم) - Possible, but requires manual iteration/comparison. Yes (نعم) - Possible, but requires manual iteration/comparison. No (لا) - **Impossible**, sets inherently prevent duplicates. Yes (نعم) - Duplicates are allowed in values, but keys must be unique.
3. Iterating Through Items Yes (نعم) - Standard `for item in list:` Yes (نعم) - Standard `for item in tuple:` Yes (نعم) - Standard `for item in set:` (Order is arbitrary). Yes (نعم) - Iterates over keys by default.
4. Unpacking Yes (نعم) - E.g., `a, b, c = my_list` (If lengths match). Yes (نعم) - E.g., `a, b, c = my_tuple`. Yes (نعم) - E.g., `a, b, c = my_set` (If size matches, order arbitrary). Yes (نعم) - Unpacks keys: `a, b = my_dict`.
5. Sorting Elements Yes (نعم) - Using `list.sort()` (In-place) or `sorted(list)`. Yes (نعم) - Using `sorted(tuple)` (Returns a new list). Yes (نعم) - Using `sorted(set)` (Returns a new list). Yes (نعم) - Using `sorted(dict.items())` (Sorts by key or value).
6. Removing Duplicates Yes (نعم) - By converting to a set and back to a list: `list(set(my_list))`. Yes (نعم) - By converting to a set and back to a tuple: `tuple(set(my_tuple))`. No (لا) - Not applicable, duplicates never exist. No (لا) - Duplicates only exist in values; keys must be unique.
7. Adding Items (Mutability) Yes (نعم) - Mutable: `list.append()`, `list.extend()`. No (لا) - **Immutable**: Must create a new tuple (`tuple + (item,)`). Yes (نعم) - Mutable: `set.add()`, `set.update()`. Yes (نعم) - Mutable: `dict['new_key'] = value`.
8. Checking if Item Exists Yes (نعم) - Using `if item in my_list:`. Yes (نعم) - Using `if item in my_tuple:`. Yes (نعم) - Using `if item in my_set:` (Extremely fast $\mathcal{O}(1)$). Yes (نعم) - Checks for key existence: `if key in my_dict:` (Fast $\mathcal{O}(1)$).
