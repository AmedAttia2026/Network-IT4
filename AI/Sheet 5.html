Part A - True / False Questions

Instructions: Write "T" for True or "F" for False.

1. NumPy is mainly used for creating and manipulating numerical arrays in Python.
(T)
2. Pandas Series can hold multiple columns of data.
(F) - **Rationale:** A Pandas **Series** holds one column; a **DataFrame** holds multiple columns.
3. The `reshape()` function in NumPy changes the dimensions of an array.
(T)
4. DataFrames in Pandas can be thought of as 2D labeled data structures similar to an Excel sheet.
(T)
5. `flatten()` in NumPy converts a multidimensional array into a 1D array.
(T)
6. SciPy is mainly used for symbolic mathematics and logic programming.
(F) - **Rationale:** SciPy is for scientific computing (e.g., optimization, statistics, integration); **SymPy** is used for symbolic math.
7. The modern replacement for `pandas.Panel` is `xarray.Dataset`.
(T) - **Rationale:** `pandas.Panel` was deprecated, and the recommended replacements for 3D+ data are `xarray` or hierarchical indexing.
8. Matplotlib is used for creating visualizations like line and bar charts.
(T)
9. In a NumPy 2D array, elements are accessed using two indices: `[row][column]`.
(T) - **Rationale:** While `[row, column]` is preferred, `[row][column]` also works in NumPy.
10. Pandas DataFrames can only contain numeric data types.
(F) - **Rationale:** DataFrames can hold mixed data types (strings, booleans, dates, numbers, etc.).

1. NumPy Array Creation: Create a 2D NumPy array with values from 1 to 9 and reshape it into 3x3.
import numpy as np

# Create array from 1 up to (but not including) 10
arr = np.arange(1, 10)
# Reshape into 3 rows and 3 columns
arr_3x3 = arr.reshape(3, 3)
print(arr_3x3)
# Output:
# [[1 2 3]
# [4 5 6]
# [7 8 9]]
2. Flatten Array: Write a program to flatten a 3D NumPy array into 1D.
import numpy as np

# Example 3D array (2x2x2)
arr_3d = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])

# Flatten the array
arr_1d = arr_3d.flatten()
print(arr_1d)
# Output: [1 2 3 4 5 6 7 8]
3. Pandas Series Example: Create a Pandas Series of student names and display the index and data type.

names = ["Ali", "Sara", "Omar"]

print("Series Index:")

print("\\nSeries Data Type:")
print(student_series.dtype)

4. DataFrame Example: Create a Pandas DataFrame for 3 students with their names, ages, and grades.
import pandas as pd

data = {
'Name': ['Ali', 'Sara', 'Omar'],
'Age': [22, 21, 23],
'Grade': [90, 95, 88]

5. Matplotlib Plot: Plot a simple line graph of $y = x^2$ using Matplotlib.
import matplotlib.pyplot as plt
import numpy as np

# Create x and y data
x = np.linspace(-5, 5, 100) # 100 points between -5 and 5
y = x**2

plt.figure(figsize=(6, 4))
plt.plot(x, y, label='y = x^2', color='red')
plt.title('Parabolic Function Plot')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.grid(True)
plt.legend()
plt.show()
